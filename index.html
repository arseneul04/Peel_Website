Safety

PEEL is designed with obvious user controls. This website is static and does not collect personal data by itself. Any account, matching, and messaging logic lives inside the app and its backend.

  • Keep reporting and blocking visible and easy.
  • Encourage respectful conversations with clear guidelines.
  • Only store what you need to run the app.
